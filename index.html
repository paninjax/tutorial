Tutorial Git - GitHub

Comandos para usar uma vez na vida:

  • git config --global user.name “...":

    • Registrar o nome

  • git config --global user.email “princesa@gmail.com":

    • email

Comandos que mais vamos usar:

  • git init:

    • avisa o git para ficar de olho na pasta

  • git status:

    • mostra o que foi modificado, adicionado, ou deletado da pasta

  • git add nome-do-arquivo.html:

    • vamos colocar na caixinha essa versão do arquivo

  • git add .:

    • vamos colocar na caixinha todos os arquivos

  • git commit -m “inclui botão vermelho”:

    • vamos colocar a etiqueta na caixinha

  • git log:

    • mostra a timeline do nosso projeto c/ todos os commits que fizemos

Comandos que mais vamos usar:

  • git checkout -- arquivo.txt:

    • para voltar para a versão do último commit

  • git branch nome-do-branch:

    • - para criar um branch (um espacinho separado para trabalharmos, e depois que estiver tudo lindo, juntamos na nossa master novamente)

  • git checkout nome-do-branch:

    • para trocarmos de branch (antes nós só criamos o branch!)

  • git merge nome-do-branch -:

    • para juntar o nosso branch c/ nosso projeto principal.
